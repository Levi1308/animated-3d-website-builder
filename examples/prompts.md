# Example Prompts

Each prompt names a brand, stack, signature interaction, assets, motion style, responsive behavior, and validation expectation.

1. Build a Lithos-style geology hero for "Strata Archive" in Vite React + TypeScript + Tailwind. Use two provided rock-face image URLs, a cursor-following circular spotlight reveal, Inter + Playfair Display typography, staggered heading reveal, fixed glass nav, responsive bottom copy blocks, and reduced-motion fallback. Run lint and build.

2. Build a VANGUARD-style creative agency hero for "Northline Studio" in Next.js + Tailwind. Use the provided looping MP4 and poster image, fullscreen overlaid content, editorial typography, desktop nav, mobile fullscreen overlay menu with `lucide-react`, CTA row, stats row, and staggered fade-up motion. Validate mobile layout and build.

3. Build a Veldara-style immersive landing page for "Auralis Residences" in Vite React. Use a fixed fullscreen scroll-scrubbed video background, lightweight particle canvas overlay, hero fade on scroll, fixed reveal cards with CSS masks, static fallbacks, and reduced-motion behavior. Run typecheck and build.

4. Create a luxury architecture studio landing page for "Atelier Vale" in React + Tailwind. Use provided exterior/interior photos, parallax image masks, precise editorial spacing, a sticky transparent nav, restrained scroll reveals, mobile-first layout, and no extra dependencies beyond Framer Motion if already installed. Validate mobile overflow.

5. Create a premium AI product page for "Signal Loom" in Next.js App Router. Use animated glass nav, procedural dashboard panels, scroll-triggered feature reveals, interactive cards, `lucide-react` icons, dark high-contrast UI, and a static fallback for reduced motion. Run lint, typecheck, and build.

6. Create a portfolio homepage for "Mira Chen" in Vite React + TypeScript. Use a procedural React Three Fiber hero object, Drei lighting helpers, project cards with keyboard focus states, subtle pointer parallax, a 2D fallback, and mobile-safe canvas sizing. Run build and inspect for blank canvas states.

7. Improve the existing boring hero for "Helio Bank" into a cinematic editorial landing page. Keep the current React/TypeScript stack and copy, add a stronger visual hierarchy, one signature parallax image mask, concise CTAs, responsive stats, purposeful Framer Motion reveals, and no routing changes. Run existing validation scripts.

8. Add a premium mobile fullscreen menu to the existing "Vessel Works" landing page. Use `lucide-react` Menu/X icons, accessible labels, body scroll lock, staggered menu transitions, visible focus states, desktop nav preservation, and Escape-to-close when practical. Run lint and test keyboard behavior.

9. Add a canvas particle layer behind the "Orbit Labs" hero in Vite React. Keep particles lightweight, scale count by viewport area, resize with DPR, use `pointer-events: none`, disable continuous animation for reduced motion, and preserve the existing hero content. Run build and check mobile performance.

10. Add a scroll-reveal card section for "Canyon Systems" using CSS masking. Use a fixed card stage on desktop, stacked fallback on mobile, horizontal mask reveal on desktop, vertical reveal on mobile, opacity fade by scroll progress, and pointer events only for active cards. Run typecheck and build.

11. Build a full-screen product launch page for "Forma One" in Next.js + Tailwind. Use supplied product images, layered depth, fixed nav, image zoom on load, hover-responsive CTA buttons, a specs strip, and a responsive layout that shows the next section hint below the fold. Run lint and build.

12. Create a cinematic event microsite for "Nocturne 2027" in Vite React. Use provided background video, custom font imports, masked ticket CTA, countdown stats, mobile overlay nav, reduced-motion poster fallback, and restrained fade/scale entrance motion. Validate console, mobile layout, and build.
